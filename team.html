About the Project

🎯 What We Built

ActivityHub is a campus activity calendar that lets students create, discover, and join activities. From study groups to sports games, everything is in one place with real-time RSVP tracking and category filters.

⚙️ Tech Stack

Built with vanilla web technologies as required — no frameworks used. Runs on a Raspberry Pi 0 2W with a PHP scripting engine and MariaDB database.

HTML5 CSS3 Vanilla JS PHP MariaDB RPi 0 2W
